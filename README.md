# vim-config

Конфиг располагается в `~/.config/nvim/init.lua`

Заточен под golang

Менеджер плагинов - lazy.nvim

Url: https://github.com/folke/lazy.nvim

Плагины скачиваются в `~/.local/share/nvim/lazy/`

Назначение плагинов и хоткеев подписаны в комментариях.

### Схема установки

1. Установить nvim
2. Установить golang
3. Установить git
4. Скопировать конфиг init.lua в `~/.config/nvim/init.lua`
5. При первом запуске nvim произойдёт установка плагинов. 
6. Установить lsp (языковой сервер) `:Mason`. Необходимо выбрать как минимум gopls, можно ещё lua для конфига.
7. Если mason не установит gopls автоматически вызвать команду `go install golang.org/x/tools/gopls@latest`
8. Связать буфер обмена с системным необходимо установкой xclip

Команда `:Lazy` выведет окно для взаимодействия с менеджером плагинов, где можно обновить, установить, удалить плагины и выполнить прочие действия
